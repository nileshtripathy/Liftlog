# LiftLog

## What It Is

LiftLog is a web application that helps me record and track my gym workouts.

## Problem It Solves

I often forgot the weight and reps from my previous gym sessions. LiftLog stores every workout so I can track my progress and continue lifting with confidence.

## Features

- Create workouts
- View workout history
- Edit workouts
- Delete workouts
- Dashboard statistics
- Dark mode
- Search workouts
- Toast notifications
- Responsive design
